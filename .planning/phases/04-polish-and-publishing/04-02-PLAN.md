  <name>Task 1: Rewrite example app with streaming simulation</name>
  <files>example/lib/main.dart</files>
  <action>
Rewrite example/lib/main.dart to be a complete, polished demonstration of the anchor-on-send behavior. The example must clearly show the package's value proposition when someone views it on pub.dev.

**Data model:**
Create a simple `ChatMessage` class (inline in main.dart) with:
- `String text` (mutable for AI messages that grow during streaming)
- `bool isUser` (true for user messages, false for AI)

**State:**
- `List<ChatMessage> _messages` — starts with 2-3 seed messages so the list is not empty
- `AiChatScrollController _controller`
- `TextEditingController _textController` — for the input field
- `Timer? _streamTimer` — for the streaming simulation
- `bool _isStreaming` — local flag to disable send button during streaming

**UI layout (Scaffold):**
- AppBar with title "ai_chat_scroll Demo"
- Body: `AiChatScrollView` with `controller`, `itemCount: _messages.length`, `itemBuilder` that returns styled message bubbles
- Message bubble styling: user messages right-aligned with a blue/indigo background, AI messages left-aligned with a grey background. Use `Container` with `BoxDecoration(borderRadius, color)` and `Padding`. Keep it simple — 15-20 lines for the bubble widget, not a separate file.
- Bottom: Row with `TextField` (expanded) + send `IconButton`. Wrap in `SafeArea` and `Padding`.

**Send flow (_sendMessage):**
1. Read text from `_textController`, clear it
2. Add `ChatMessage(text: userText, isUser: true)` to `_messages`
3. Add `ChatMessage(text: '', isUser: false)` as the empty AI placeholder
4. `setState(() {})`
5. Call `_controller.onUserMessageSent()`
6. Set `_isStreaming = true`
7. Start a `Timer.periodic(Duration(milliseconds: 50))` that appends one word at a time from a canned response string to the AI message's `text` field, calling `setState` on each tick
8. The canned response should be a realistic AI-style paragraph, e.g.: "That's a great question! The anchor-on-send pattern works by snapping your message to the top of the viewport when you send it. As this response streams in word by word, notice how your message stays fixed at the top while the text grows below. The filler space shrinks dynamically to maintain this position. You can try scrolling down manually during streaming to break the anchor. Pretty neat, right?"
9. When all words are appended, cancel the timer, call `_controller.onResponseComplete()`, set `_isStreaming = false`, `setState`

**Seed messages:**
```dart
ChatMessage(text: 'Welcome to the ai_chat_scroll demo!', isUser: false),
ChatMessage(text: 'Try sending a message to see the anchor behavior.', isUser: false),
```

**Dispose:** Cancel `_streamTimer`, dispose `_controller` and `_textController`.

**Do NOT:**
- Import any packages beyond flutter/material.dart and ai_chat_scroll
- Use any state management packages
- Add complex animations or transitions
- Create multiple files — everything in main.dart
