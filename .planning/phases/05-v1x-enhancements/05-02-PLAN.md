  **Widget changes (ai_chat_scroll_view.dart):**

  1. Add a field `double _lastViewportDimension = 0.0` to track the previous viewport height.

  2. In `_executeAnchorJump()` (end of anchor pipeline), after setting `_lastMaxScrollExtent`, also capture: `_lastViewportDimension = pos.viewportDimension`.

  3. In the existing `ScrollMetricsNotification` handler (the `onNotification` callback in `build()`), add viewport dimension change detection:
     ```dart
     onNotification: (notification) {
       final currentVD = notification.metrics.viewportDimension;
       if (_anchorActive && _lastViewportDimension > 0.0 && currentVD != _lastViewportDimension) {
         _onViewportDimensionChanged(currentVD);
       }
       _onScrollChanged();
       return false;
     }
     ```

  4. Add `_onViewportDimensionChanged(double newViewportDimension)`:
     ```dart
     void _onViewportDimensionChanged(double newViewportDimension) {
       final delta = newViewportDimension - _lastViewportDimension;
       _lastViewportDimension = newViewportDimension;

       // delta > 0 means viewport grew (keyboard closed) -> filler grows
       // delta < 0 means viewport shrunk (keyboard opened) -> filler shrinks
       _fillerHeight.value = math.max(0.0, _fillerHeight.value + delta);

       // After filler change, re-anchor to keep sent message at viewport top
       SchedulerBinding.instance.addPostFrameCallback((_) {
         if (!mounted || !_scrollController.hasClients) return;
         final pos = _scrollController.position;
         _scrollController.jumpTo(pos.maxScrollExtent);
         _lastMaxScrollExtent = pos.maxScrollExtent;
       });
     }
     ```

  5. Also update `_lastViewportDimension` in `_recomputeFiller()` to keep it current during normal streaming filler updates (when content grows, viewport doesn't change, but we want the latest value):
     ```dart
     void _recomputeFiller() {
       if (!_anchorActive || !_scrollController.hasClients) return;
       final pos = _scrollController.position;
       _lastViewportDimension = pos.viewportDimension; // keep current
       // ... existing growth logic ...
     }
     ```
